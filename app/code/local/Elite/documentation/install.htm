DISABLE your Magento cache before installing any new modules.
Unzip the Elite/ folder to app/code/local/, so you will have app/code/local/Elite.
See Also: Configuration Guide
  • Open app/code/local/Elite/Vaf/
    Rename config.default.ini to config.ini
  • Open app/Mage.php and paste the following ( right at the beginning of the file after the <?php tag ) 
     require_once( 'code/local/Elite/Vaf/bootstrap.php' );
    Note: This needs to be repeated whenever you upgrade Magento. Just refer to the "upgrading magento" link on the table of contents.
Copy Elite/install-vaf.php to Magento's web root. Load it in your browser. Press GO to have the database created and the design files copied. The check boxes allow you to choose which actions to omit if any. The red/green boxes show you if your file permissions are sufficient for installation.
